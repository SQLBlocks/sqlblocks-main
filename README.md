# Sqlblocks

> Scratch-like visual database client for learners, built on web technologies


## Dev

```
$ npm install
```

### Run

```
$ bozon start
```

### Package

```
$ bozon package
```

Builds the app for OS X, Linux, and Windows, using [electron-builder](https://github.com/electron-userland/electron-builder).
